Bot de Telegram

📱 ¿Qué es?​

Bot inteligente que entiende lenguaje natural. Accesible desde tu móvil o cualquier dispositivo con Telegram.

💬 Ejemplos de Uso​

Crear Tareas​

Usuario: "crear tarea de revisar código"
Bot: ✅ Tarea creada: "revisar código"

Usuario: "agregar tarea importante: documentación"
Bot: ✅ Tarea creada: "documentación" (Prioridad: Alta)

Usuario: "nueva tarea de baja prioridad: arreglar typos"
Bot: ✅ Tarea creada: "arreglar typos" (Prioridad: Baja)

Consultar Tareas​

Usuario: "¿cuáles son mis tareas?"
Bot: Tienes 5 tareas:
     1. Revisar código (Alta)
     2. Documentación (Alta)
     3. Deploy (Media)
     4. Arreglar typos (Baja)
     5. Tests (Media)

Usuario: "¿qué tareas de alta prioridad tengo?"
Bot: Tienes 2 tareas de alta prioridad:
     1. Revisar código
     2. Documentación

Usuario: "¿cuántas tareas completadas tengo?"
Bot: Tienes 3 tareas completadas.

Cambiar Estado​

Usuario: "marcar revisión de código como completada"
Bot: ✅ Tarea "Revisar código" marcada como completada

Usuario: "mi tarea de documentación está en progreso"
Bot: ✅ Estado actualizado a "En Progreso"

Asignar Sprint​

Usuario: "asignar revisar código a sprint 2"
Bot: ✅ Tarea asignada a Sprint 2

Usuario: "¿qué tareas tengo en sprint 1?"
Bot: Sprint 1 tiene estas tareas:
     - Deploy
     - Arreglar typos

🤖 Características​

✅ Lenguaje Natural​

No necesitas memorizar comandos, escribe como si hablaras:

❌ ANTES: /crear "titulo" --prioridad alta
✅ AHORA: "crear tarea importante"

✅ Sincronización Real​

Los cambios en Telegram se ven en la Web y viceversa:

Web: Creo tarea
  ↓
Telegram: "mis tareas" → ve la nueva

Telegram: Cambio estado
  ↓
Web: Panel actualiza automáticamente

✅ Contexto Completo​

El bot entiende contexto:

Usuario: "marcar la primera como completada"
Bot: ✅ "Revisar código" completada
(Entiende que se refiere a la primera de la lista)

✅ Disponible 24/7​

Accede desde tu móvil en cualquier momento:

Antes de iniciar: "¿qué tareas tengo hoy?"
Durante: "marcar completada"
Después: "resumen del día"

📲 Cómo Usar​

Empezar​

1. Busca en Telegram: @MiOraclePlannerBot
2. O usa este link: https://t.me/MiOraclePlannerBot
3. Click en "Empezar"
4. Escribe naturalmente

Interacción Natural​

"crear tarea importante"
"mis pendientes"
"completa esto"
"¿cuándo es el sprint?"
"mostrar todo"

Sin Comandos Especiales​

❌ NO necesitas memorizar:
/crear /edit /delete /show /status

✅ SOLO escribe normalmente:
"crea una tarea"
"edita esa"
"quita eso"
"muestra todo"
"¿cómo está?"

🎯 Casos de Uso Ideales​

SituaciónUso
Mientras viajoCrear/consultar tareas rápidamente
ReuniónPreguntar estado sin abrir laptop
Andar en la calleAnotar idea rápida
NotificacionesRecibir actualizaciones en tiempo real
AccesibilidadUsar voz o texto según necesidad

⚠️ Limitaciones​

  • Vista limitada por formato de chat
  • No puedes ver todas las tareas a la vez
  • Mejor para consultas puntuales que para visualización completa

Solución: Usa Web UI para vista completa, Telegram para acceso rápido

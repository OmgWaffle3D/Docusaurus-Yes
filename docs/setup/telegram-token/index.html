Obtener Token de Telegram

📱 Paso a Paso​

1. Abre BotFather​

En Telegram, busca o haz click aquí: @BotFather

2. Crear Nuevo Bot​

Envía el comando:

/newbot

3. Responder Preguntas​

BotFather te preguntará:

📝 Give your new bot a name.
For example, my awesome bot.

Responde: El nombre que prefieras (ej: MiOraclePlannerBot)

🔹 Good. Now let's choose a username for your bot.
It must end in `bot`. For example, tetris_bot or tetris_testing_bot.

Responde: El nombre único (ej: mi_oracle_planner_bot)

4. Copiar el Token​

BotFather responderá con algo como:

🎉 Done! Congratulations on your new bot. 
You'll find it at t.me/mi_oracle_planner_bot. 
You can now add a description, about section and profile picture for your bot, 
see /help for a list of commands.

Use this token to access the HTTP API:
123456789:ABCDefGHIJKlmnoPQRstuvwxyz

Keep your token secure and store it safely!

Copia: 123456789:ABCDefGHIJKlmnoPQRstuvwxyz

🔐 Protege Tu Token​

⚠️ IMPORTANTE: El token es como una contraseña

  • No lo publiques en GitHub público
  • No lo compartas en redes sociales
  • No lo incluyas en commits

✅ ¿Qué Hacer Ahora?​

Usa el token en:

telegram.bot.token=123456789:ABCDefGHIJKlmnoPQRstuvwxyz

Habilitar LLM (Opcional)

Para respuestas más inteligentes con procesamiento de IA, integra OpenAI o similar.

🚀 Activar Procesamiento con IA​

En application.properties:

# Habilitar IA
agent.ai.enabled=true

# Configuración de OpenAI
agent.ai.base-url=https://api.openai.com/v1
agent.ai.api-key=sk-tu-clave-aqui
agent.ai.model=gpt-4o-mini

📌 Obtener API Key de OpenAI​

1. Crear Cuenta​

Visita: https://platform.openai.com

2. Ir a API Keys​

Menu → API Keys → Create new secret key

3. Copiar la Clave​

Guárdala en un lugar seguro

4. Agregar a Propiedades​

agent.ai.api-key=sk-proj-xxx...

💰 Costo​

  • Gratis para empezar: Prueba de $5 iniciales
  • Después: Pagas por uso (usualmente muy económico)
  • Modelos baratos: gpt-4o-mini es ideal para este proyecto

🤖 Modelos Disponibles​

ModeloCostoVelocidadRecomendado
gpt-4o💰💰💰NormalNo
gpt-4o-mini💰Rápido✅ Sí
gpt-3.5-turbo💰Muy rápido✅ Sí

⚠️ Sin IA (Por Defecto)​

Si no configuras esto, el bot funcionará con:

  • ✅ Comandos predefinidos
  • ✅ Lógica de reglas
  • ❌ Sin procesamiento natural de lenguaje
# Para desabilitar
agent.ai.enabled=false

🔐 Protege tu API Key​

  • No la publiques en GitHub
  • No la compartas en redes sociales
  • Considera usar variables de entorno
export OPENAI_API_KEY=sk-proj-xxx

Entonces en properties:

agent.ai.api-key=${OPENAI_API_KEY}
